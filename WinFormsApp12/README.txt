Приложение WinForms, отрисовывающее график функции - спираль Архимеда, с последующей анимацией прохождения геометрической фигуры по этому графику. 
Графическая отриcовка реализована с помощью библиотеки System.Drawing.
Анимация реализована через цикл For с замедлением потока через System.Threading.
Основной файл с кодом находится по пути: /WinFormsApp6/Form1.cs
