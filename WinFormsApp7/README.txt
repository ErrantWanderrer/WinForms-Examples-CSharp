Приложение WinForms, реализующее двусвязный не кольцевой список, в котором хранятся объекты класса Vector.cs.
Графическая отриcовка реализована с помощью библиотеки System.Drawing.
Основной файл с кодом находится по пути: /WinFormsApp6/Form1.cs
Класс объектов Vector.cs находится по пути: /WinFormsApp6/Vector.cs
Класс, реализующий двусвязный не кольцевой список находится по пути: /WinFormsApp6/VecList.cs
