Приложение WinForms для расчета величин 4-х геометрических фигур (точка/прав.пятиугольник/прям-уг.треугольник/прямоугольник).
Параметры для каждого объекта введены заранее. 
Рассчеты для некоторых показателей реализованы двумя способами - через виртуальный и невиртуальный методы.
Основной файл с кодом находится по пути: /WinFormsApp5/Form1.cs
