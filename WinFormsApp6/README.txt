Приложение WinForms для отрисовки двух геометричесикх фигур с вводом координат и длин сторон.
Реализовано с помощью библиотеки System.Drawing.
Основной файл с кодом находится по пути: /WinFormsApp6/Form1.cs
Интерфейс и два производных класса от него находятся по пути: /WinFormsApp6/Interface1.cs, /WinFormsApp6/Class1.cs, /WinFormsApp6/Class2.cs
Диалоговое окно, наследуемое от Form1.cs находится по пути: /WinFormsApp6/Dialog.cs
